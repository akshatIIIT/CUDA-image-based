This project focuses on building a GPU-based image processing pipeline using CUDA, where a blur filter is applied to a large set of images using parallel execution on the GPU. The main objective was to demonstrate how GPU computing can significantly improve the performance of image processing tasks, especially when working with large datasets. For implementation, a CUDA kernel called blurKernel was used to apply a 3×3 averaging filter, where each thread processes one pixel in parallel, allowing multiple pixels to be handled simultaneously. The program loads more than 500 images from the images/input directory and processes them in a single run using GPU acceleration. CUDA features like cudaMalloc were used for device memory allocation, and cudaMemcpy was used for transferring data between host and device. The results showed that all images were successfully processed with a clear blur effect, and the use of GPU parallelism noticeably improved performance for pixel-level operations. During development, some challenges were faced, such as handling empty images, managing file paths correctly for batch processing, and ensuring proper memory allocation. Overall, this project demonstrates the effectiveness of GPU acceleration and highlights the advantages of parallel computing for handling large-scale image processing tasks.
